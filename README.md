# Spelling Quiz! ✏️

## Overview
An interactive spelling quiz application designed to help students practice and improve their spelling skills through multiple learning modes.

## Features
- **Three Quiz Modes:**
  - 💡 **Hint Mode** - Read a clue and spell the word
  - 🔊 **Listen & Spell** - Hear the word spoken aloud (just like the real test!)
  - 🌟 **Both Together** - Get both a hint and hear the word

- **Performance Tracking:**
  - Real-time score display
  - Visual progress bar
  - Detailed breakdown of results (1st try, 2nd try, 3rd try, missed)
  - Practice list for words you missed

- **User-Friendly Interface:**
  - Beautiful, colorful design perfect for students
  - Animated interactions and visual feedback
  - Try dots to show attempt progress
  - Encouraging messages throughout

## How to Use
1. Open `index.html` in your web browser
2. Select your preferred quiz mode
3. Read the hint or listen to the word (depending on mode)
4. Type your spelling in the input field
5. Click "Check ✓" to verify your answer
6. You get up to 3 attempts per word
7. View your results and performance breakdown at the end

## Word List
The quiz includes 20 spelling words commonly found in elementary school assessments:
- pounce, placed, dice, cents, price, space, mice, office, wage, age, gyms, giant, changes, message, pages, you, road, peace, giraffe, peaceful

## Technical Details
- Built with vanilla HTML, CSS, and JavaScript
- Uses Puter AI for text-to-speech functionality
- Responsive design works on desktop and mobile devices
- No external dependencies required (except for optional TTS)

## Browser Support
Works best on modern browsers (Chrome, Firefox, Safari, Edge)

## Getting Started
Simply download or clone the repository and open `index.html` in your browser. No installation or setup required!
